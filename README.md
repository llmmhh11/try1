# Aurora Player — 粒子音乐播放器

一个基于 Three.js + Web Audio API 的沉浸式音乐播放器，纯黑空间中彩色粒子随音乐律动，播放时汇聚成专辑封面并以波浪形式立体浮动。

## ✨ 核心特性

### 视觉设计
- **纯黑空间** — 全黑背景，营造深邃沉浸感
- **悬浮玻璃态控制** — 左右两侧毛玻璃面板，功能按键漂浮于空间中
- **中心焦点光晕** — 微妙呼吸光晕引导视觉焦点
- **8000 粒子系统** — GPU 级渲染，AdditiveBlending 发光效果

### 粒子动画
- **随机分布 → 封面汇聚** — 播放音乐时，粒子从球形随机分布平滑过渡为专辑封面方形
- **对角线波浪传播** — 波浪自左上角至右下角沿对角线方向传播
- **Y轴 + Z轴立体浮动** — 粒子在垂直方向和深度方向（向用户）同时浮动，形成立体波浪
- **音频驱动脉动** — 低频/中频频谱实时驱动粒子脉动与发光

### 音频功能
- **本地文件导入** — 支持导入任意音频文件（MP3/WAV/FLAC 等）
- **播放列表管理** — 多文件导入自动生成播放列表
- **频谱分析** — Web Audio API AnalyserNode 实时分析
- **完整播放控制** — 播放/暂停/上一首/下一首/进度跳转/音量/静音

## 🎮 使用方法

### 基本操作
1. 点击左侧面板的 **导入按钮**（上传图标）加载音乐文件
2. 粒子自动汇聚成默认渐变封面并开始播放
3. 点击 **封面按钮**（图片图标）可自定义专辑封面
4. 使用右侧面板调节波浪强度和粒子密度

### 键盘快捷键
| 按键 | 功能 |
|------|------|
| `空格` | 播放/暂停 |
| `←` | 后退 5 秒 |
| `→` | 前进 5 秒 |
| `Shift + ←` | 上一首 |
| `Shift + →` | 下一首 |
| `M` | 静音切换 |

## 🏗️ 技术架构

### 技术栈
- **Three.js 0.160.0** — WebGL 粒子渲染，自定义 ShaderMaterial
- **Web Audio API** — 音频播放与实时频谱分析
- **Canvas API** — 专辑封面像素提取
- **ES6 Modules** — 模块化代码组织
- **纯 CSS** — 玻璃态 UI，无框架依赖

### 文件结构
```
播放器/
```

### 着色器核心逻辑

**顶点着色器**实现三大功能：
1. `mix(aRandomPos, aTargetPos, delayedT)` — 随机位置到封面位置的平滑过渡
2. `sin(diagProgress * 10.0 - uTime * 2.5)` — 对角线波浪相位计算
3. `pos.y += yWave; pos.z += zWave;` — Y轴上下浮动 + Z轴向用户浮动

**片元着色器**实现软边圆形粒子 + 中心发光效果。
