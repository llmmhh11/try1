# Aurora 粒子音乐播放器 — 项目概览

## 📋 项目简介

为团队打造的一款沉浸式音乐播放器，采用 Three.js 360K 粒子系统 + Web Audio API 频谱分析，实现纯黑空间中粒子随音乐律动、汇聚为像素级清晰专辑封面的视觉体验。

## ✅ 完成内容

### 1. 视觉设计
- **纯黑背景**：全黑空间 + 径向渐变深度感
- **悬浮控制面板**：左右两侧毛玻璃态面板（blur + saturate）
- **中心焦点光晕**：呼吸动画引导视觉焦点
- **完整 UI**：品牌栏、进度条、播放列表、Toast 提示
- **视口自适应**：所有尺寸使用 vw/vh/vmin 相对单位，任意窗口保持比例

### 2. 粒子系统（核心）
- **360,000 粒子** GPU 渲染，Custom ShaderMaterial
- **像素级封面映射**：600×600 = 360K 像素 1:1 映射，人物五官清晰可辨
- **对角线波浪**：波浪自左上角至右下角传播，速度适中（5.2s周期）
- **立体浮动**：Y轴上下 + Z轴向用户双重浮动
- **圆形粒子美学**：零间隙显示 + 密度亮度补偿 + 8%锐利边缘

### 3. 音频引擎
- Web Audio API 频谱分析（低频70% + 中频30%加权）
- 文件导入、播放列表、完整播放控制
- 键盘快捷键支持

## 🛠️ 技术栈
- Three.js 0.160.0（本地化，离线可用）
- Web Audio API + AnalyserNode
- 自定义 GLSL ShaderMaterial（顶点+片元着色器）
- CSS vw/vh/vmin 视口自适应
- 纯 HTML/CSS/JS，无框架依赖

## 📊 关键参数
| 参数 | 值 | 说明 |
|------|-----|------|
| 粒子数 | 360,000 | 600×600 分辨率 1:1 像素映射 |
| 封面面积 | 14 世界单位 | 占垂直 FOV 86.6%，匹配 UI 可用空间 |
| 粒子混合 | NormalBlending | 显示本色，不过曝 |
| uPointScale | 28 | 圆形粒子对角线全覆盖，零间隙 |
| 波浪速度 | uTime×1.2 | 5.2秒完整周期 |

## 🚀 使用方式
1. 启动本地服务器（localhost:8765）
2. 点击左侧导入按钮加载音乐文件
3. 粒子自动汇聚成封面并播放
4. 可自定义专辑封面、调节波浪强度和粒子密度
5. 快捷键：空格播放/暂停，←→快退快进，Shift+←→切歌，M静音
